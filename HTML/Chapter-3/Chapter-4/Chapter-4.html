  1. WRITING LINKS

    • Links are created using the <a> </a> element
    • The <a> is the anchor element.
    • The <a> is the anchor element.
    • In HTML5, the <a> tag is always a hyperlink, but if it has no href attribute, it is only a --- placeholder for a hyperlink.
    Attributes of <a> elements are;

      href:
      Specifies the URL of the page the link goes to

      downloads:
      Specifies that the target will be downloaded when a user clicks on the hyperlink. The value of this element will be “filename”

      target:
      Specifies where to open the linked document. Its values are;

      • "_blank" :Opens the linked document in a new window or tab
      • "parent" :Opens the linked document in the same frame as it was clicked (this is default)
      • "self" :Opens the linked document in the parent frame
      • "top" :Opens the linked document in the full body of the window
      • "framename" :Opens the linked document in a named frame

    link text
    - The text between opening <a> and closing </a> tags is known as link text.

  2. LINKING TO OTHER SITES

    • ABSOLUTE URLs
      • URL – Uniform Resource Locator.
      • Absolute URLs – are the links of different websites.
      • An absolute URL starts with domain name and followed by the path to a specific page.If no page is specified the site will display the home page.
            • <p>
              <ul>
                <li> <a href=”http://www.empireonline.com”> Empire</a></li>
                <li> <a href=”http://www.metacritic.com”> Metacritic</a></li>
              </ul>
            </p>

  3. LINKING TO OTHER PAGES ON THE SAME SITE

    • RELATIVE URLs

      • When do you link other pages of the same site you do not need to specify the domain name.A shorthand can be used known as Relative URL.
      If all the pages are in the same folder then “href attribute” value would be the file name only.

          <p>
            <ul>
              <li> <a href=”index.html”>Home</a></li>
              <li> <a href=”about us.html”>About</a></li>
            </ul>
          </p>

      Index html is a root directory.
      Index html are the home pages.
      Every parent, child, and grandchild folder has its own home page / index.html
      If you link into a page from two different pages you might need to write two different relative URLs.

    • RELATIVE LINK TYPES

      • SAME FOLDER

        • Here we need to use file name only.
        <a href=”filename.html”> Reviews</a>

      • CHILD FOLDER

        • When file is in child folder,first to write child folder name and after forward slash write file name
        <a href="child folder/filename.html”>Reviews</a>

      • GRAND CHILD FOLDER

        • When file is in grand child folder, first write child folder name then after forward slash write grand child folder name and then file name after forward slash.
        <a href="childfolder/grandchildfolder/filename.html">Reviews</a>

      • PARENT FOLDER

        • When to move parent folder from child folder, we use ../
        <a href="../filename.html">Home</a>

      • GRAND PARENT FOLDER

        • When we want to go two levels up from grand child to grand parent we use ../../ and similarly as on.
        <a href="../../filename.html">Home</a>

    Note: A forward slash will return to the home page for entire site.
    A forward slash followed by a file name will return to that file provided it is in the root directory.
    If you see the child folder name without file name web server will show the homw page.

  4. OPENING LINKS IN A NEW WINDOW

    5. To open a link into new window we use target attribute.And its value is “ _blank”.
    <a href="https://www.w3schools.com” target="_blank">

  5. LINKING TO A SPECIFIC PART OF THE SAME PAGE

    6. The id attribute specifies a unique id for an HTML element. The value must be unique within the HTML document.

    • The value of id attribute should start with a letter or an underscore.
    • No number or character can be used for id value.
    • Two ids should not have same value.
    • To link to a specific part of a same page first we have to give an id to that specific part by “id attribute e.g. <h1 id=”top”>MAIN HEADING </h1>.
    • Then we have to create a link by <a> element anywhere on the page e.g. at bottom,
      <p> <a href=”#top”>Top</a></p>.
    • The value of <a> element starts with the # symbol followed by the id attribute

  6. LINKING TO A SPECIFIC PART OF ANOTHER PAGE

    • We use the same syntax as in the same page link.
    • But in <a> element href attribute will contain the address for the page, either absolute or relative URL, followed by the # symbol and then followed by the value of id attribute.
      <a href=”Absolute or relative url/#value of id “>value of id</a>
