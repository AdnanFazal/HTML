L I S T S

There are three types of lists in HTML;

  • ORDERED LISTS

    • <ol> </ol> .Ordered list uses where sequence or order is important.
    • <ol type=”1”> or “A”> or “a”> or “I”> or “I”>. Type attribute describes how to list order.
    • <li> </li>.Every item of a list will be contained in this element

  • UNORDERED LISTS

    • <ul> </ul> In unordered list order is not important. And the change in order does not affect the page meaning.
    • <ul type=”disc”> or “circle”> or “square”>.
    • disc – It is a default attribute. It means a filled circle.
    • circle – It means an unfilled circle.
    • Square – It menas a filled square.
    • <li> </li>.Every item of a list will be contained in this element

  • DEFINITION LISTS

    • <dl> </dl> . The definition list created with <dl>. It contains a series of terms and their definitions.
    • <dt> </dt>. It contains the term of which definition is to be defined.
    • <dd> </dd>. It contains the definition of a term.
