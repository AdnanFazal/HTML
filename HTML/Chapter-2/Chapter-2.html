  • People access websites using software called a web browser
  • <title> </title> any thing written in between title tags are shown in title bar or tab
  • Tags are known as markups
  • There are two types of markups 1- Structural markups 2- Semantic markups
  • Structural markups are used to describe both headings and paragraphs
  • Semantic markups provide extra informations e.g. to give emphasize on a sentence or word

    1. STRUCTURE MARKUPS

      • <h1>heading</h1>, <h2></h2>, <h3></h3>, <h4></h4>, <h5></h5>,<h6></h6>
      • <p>paragraph</p>
      • <b> bold </b>
      • <i> italic </i>
      • <sup> superscript </sup> example 12th March (12<sup>th</sup> March)
      • <sub> subscript </sub> It is commonly used with foot notes or chemical formulas. e.g CO2
      • White space collapsing – HTML ignores white spaces and treates them as one space.
      • Empty elements – usually have only one tag e.g. <br /> & <hr />
      • <br /> It is a line break tag (this is also acceptable <br>, but not a good habbit)
        Line break tag is used to break long lines / sentences into second line.
      • <hr /> It draws horizontal line ____ (this is also acceptable <hr>, but not a good habbit)
      • CMS – Content Management System (allows a non programmer to make changes in web)
      • There are CMS and HTML editors such as Dreamweaver. They provide both a visual editor view and a code view.
      • In visual editors new paragraphs are created using the return or enter key.
      • In visual editors line breaks are created by pressing the shiftkey+return key at the same time.
      • In visual editors do not copy paste text from MS words, first to paste it on note pad or text editor and then copy it and paste it into visual editor.

    2. SEMANTIC MARKUPS

      • Semantic markups are the elements which do not affect the structure of a web page but do add some extra information to the page.
      • Semantic markups or elements are commonly used in screen reader programs and in search engines. Search engines uses your <blockquote> informations.

        1. STRONG & EMPHASIS

          • <strong> strong importance </strong>
            • It is more than the emphasis.
            • It highlights the facts or dangers of real life.
          • <em> emphasis</em> It indicates the opinion and thoughts of a person.

        2. QUOTATIONS

          • <blockquote> </blockquote> . It shows that block of text is a quotation from other webs. It is used for longer quotes.
          • <q> quote </q>. It is used for shorter quotes.

        3. ABBREVIATIONS & ACRONYMS

          • <abbr>abbreviation</abbr> In HTML5 it is used for both abbreviations and acronyms.
            <abbr title="Hyper Text Markup Language">HTML</abbr>.A title attribute is used to specify the full term.
          • <acronym>acronym</acronym>. It used to use in HTML 4 .

        4. CITATIONS & DEFINITIONS

          • <cite>citation</cite> It gives references from books, films, or research paper.
          • <dfn>defining instances</dfn>.When , first time,the new terminology, idea, or concept is explained, it is called the defining instance of it.
            The <dfn> is used to indicate the defining instance.

        5. AUTHOR DETAIL

          • <address> </address>.It contains the contact detail of the author of the page
          • <a href=”mailto: homer@example.org> homer@example.org</a>.
            The <a> tag defines a hyperlink, which is used to link from one page to another.
            The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

        6. CHANGES TO CONTENT

          • <ins> </ins> this element is used to show the content that has to be inserted into document.It is usually shown as underline.
          • <del> </del> This element is used to show deleted content, by a line through it.
          • <s> </s> This element is used to indicate that something is no longer relevant or accurate, but should not be deleted.
