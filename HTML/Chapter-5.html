  1. ADDING IMAGES

    • <img /> - This is an image tag . It is a self closing tag. It is an empty element.
    • src - This is an image attribute, which defines the source of the image.
      <<src=”image/abc.jpg”
    • alt – This attribute provide the text description of image if you can not see the image for any reason.
    • title – This attribute provides more information about the image when you hover over the image.
    • height – This attribute describes the height of the image in pixels
      height=”450”
    • width – This attribute describes the width of an image in pixels.
      width=”100”

  2. WHERE TO PLACE IMAGES IN YOUR CODE

    3. There are three different places where we can place our images;

    • BEFORE PARAGRAPH:

      • Here paragraph starts from a new line after the image.

    • INSIDE THE START OF A PARAGRAPH:

      • The first row of the text aligns with the bottom of the image.

    • IN THE MIDDLE OF THE PARAGRAPH:

      • Here image is placed between the paragraph.

    • Block Element: Block elements like <h1> & <p> are always start from a new line after image.

    • Inline Element: Inline element like <b>, <em>, and <img> sits in a block level element and do not start on a new line.

  3. ALIGN IMAGE HORIZONTALLY

    • Left align put the image on left side and text on the right side.
      <img src=”image/html.jpg” align=”left”/>

    • Right lign put the image on right side and text on the left side
      <img src=”image/html.jpg” align=”right”/>

  4. ALIGNING IMAGES VERTICALLY

    • In top alignment first line of text starts from top of the image
    • In middle alignment first line of text starts from the middle of image.
    • In bottom alignment first line of text starts from the bottom of image.

  5. RULES FOR CREATING IMAGES

    • Chose the right format of the image. Mainly jpeg, gif and png are used on the websites.
    • Save images at the right size.
    • Use the correct resolution

  6. IMAGE FORMATS

    • JPEG formats are used for images having many different colors.
    • GIF and PNG formats are used for images having few colors or the same color on large area. This is known as the flat color, commonly used for logos, illustrations and diagrams.

  7. IMAGE RESOLUTION

    • Bitmap : Bitmap is a type of image format, made up of tiny squares.JPGs, GIFs, and PNGs are belongs to bitmap type.
    • Resolution : resolution of an image is the number of tiny squares in per inch of area.Higher number of tiny squares per inch the higher the resolution.
    • For web pages resolution should be 72 ppi.The higher the resolution the larger the size of the image and more time to display.
    • In print material like books & magazines resolution is defined by tiny circles called dots per inch (dpi).

  8. VECTOR IMAGES

    • Vector images are resolution free. Logos, illustrations, & diagrams are often created in vector format.

  9. ANIMATED GIFS

    • Animated gifs show several frames of an image in sequence which shows animation.
    • Gifs are not ideal formats for displaying photographs, but for animation they are really suitable.

  10. FIGURE AND FIGURE CAPTION

    • Figure element <figure></figure> is used to contain image element <img></img> and figurecaption <figcaption></figcaption> both.
